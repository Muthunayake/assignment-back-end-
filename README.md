## Meeting Minds User Micro Service

This service provides authentication and authorization functionalities of Meeting Minds using Nest js

## Installation

```bash
1. Create `.env` file from `sampleEnv`
2. Add the relevant environment variables
3. Run `npm install`

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
